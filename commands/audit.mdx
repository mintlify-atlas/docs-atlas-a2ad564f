## Overview

The `stobo audit` command runs comprehensive SEO and AEO (Answer Engine Optimization) audits on individual URLs or entire websites. Get actionable insights about search engine visibility and AI readiness.

## Commands

### stobo audit run

Run an audit on a specific URL.

```bash
stobo audit run <url> [options]
```

<ParamField path="url" type="string" required>
  URL to audit
</ParamField>

<ParamField path="--seo-only" type="boolean" default="false">
  Run only SEO checks, skip AEO analysis
</ParamField>

<ParamField path="--aeo-only" type="boolean" default="false">
  Run only AEO checks, skip SEO analysis
</ParamField>

<ParamField path="--keyword" type="string">
  Target keyword for SEO analysis. Alias: `-k`
</ParamField>

<ParamField path="--playwright" type="boolean" default="false">
  Force Playwright for JavaScript-rendered SPAs
</ParamField>

#### Example

```bash
# Full audit (SEO + AEO)
stobo audit run https://example.com/article

# SEO-only audit with target keyword
stobo audit run https://example.com/article --seo-only --keyword "AI optimization"

# AEO-only audit
stobo audit run https://example.com/article --aeo-only

# Audit JavaScript-heavy SPA
stobo audit run https://app.example.com --playwright
```

<Info>
  **Credit cost:** SEO-only and AEO-only audits consume fewer credits than combined audits.
</Info>

```bash

```

<ParamField path="url" type="string" required>

</ParamField>

#### Example

```bash
stobo audit site https://example.com
```

<Info>

### stobo audit list

List your recent audits.

```bash
stobo audit list [options]
```

<ParamField path="--limit" type="integer" default="20">
  Maximum number of results to return. Alias: `-n`
</ParamField>

#### Example

```bash
# List last 20 audits
stobo audit list

# List last 50 audits
stobo audit list --limit 50
```

### stobo audit get

Retrieve a specific audit by its ID.

```bash
stobo audit get <audit_id>
```

<ParamField path="audit_id" type="string" required>
  Audit UUID to retrieve
</ParamField>

#### Example

```bash
stobo audit get a1b2c3d4-5678-90ab-cdef-1234567890ab
```

---

### stobo audit get-url

Get the most recent audit for a specific URL.

```bash
stobo audit get-url <url>
```

<ParamField path="url" type="string" required>
  URL to look up
</ParamField>

#### Example

```bash
stobo audit get-url https://example.com/article
```

<Note>
  This command returns the **most recent audit** for the specified URL. If you need a specific audit, use `stobo audit get` with the audit ID.
</Note>

## Understanding grades

Audits are scored on a percentage scale and assigned letter grades:

- **A** (91-100%): Excellent optimization
- **B** (76-90%): Good, minor improvements needed
- **C** (51-75%): Fair, several issues to address
- **D** (26-50%): Poor, major optimization required
- **F** (0-25%): Critical issues

---

## JSON output

All audit commands support JSON output using the global `--json` flag:

```bash
stobo audit run https://example.com --json
```

This returns structured data suitable for automation and integration.
